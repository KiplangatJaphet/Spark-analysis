# Apache Spark Data Processing Project

This project demonstrates practical big data analytics using Apache Spark (PySpark).

It covers:

- RDD transformations and actions
- DataFrame API operations (filtering, grouping, joins, aggregations)
- Data cleaning and handling null values
- Column transformations and derived metrics
- Spark SQL queries and temporary views
- Window functions for advanced analytics
- Revenue and customer spending analysis

The project showcases how Spark handles distributed data processing efficiently using RDDs, DataFrames, and Spark SQL within a real-world analytical workflow.
